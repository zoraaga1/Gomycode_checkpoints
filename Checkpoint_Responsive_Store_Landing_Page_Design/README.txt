Project: Responsive Store Landing Page  

## Objective:  
The goal of this project is to design a visually appealing and responsive landing page for an online store.  

## Design Choices:  
- **Minimalist Design**: A clean, modern look for easy navigation and a professional appearance.  
- **Color Scheme**: Neutral and muted tones to ensure readability and a premium feel.  
- **Typography**: Selected for clarity and elegance.  
- **Responsive Layout**: Used CSS flexbox and media queries to ensure a seamless experience on both desktop and mobile devices.  
- **Navigation**: Includes a mobile-friendly menu with smooth interactions.  
- **Footer**: Contains essential links and contact details for better user experience.  

## Technologies Used:  
- HTML  
- CSS (with SCSS for maintainability)  
- Responsive Design using Media Queries  

## Instructions:  
To run this project, open the `index.html` file in a browser.
